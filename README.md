## Installation

To install the discharge_docs package use:

```{bash}
pip install -e .
```

## Deploying to PositConnect

To deploy to PositConnect make sure to have installed the requirements and run:
```{bash}
. .env
. deploy.sh
```
Choose the desired deployment options.
